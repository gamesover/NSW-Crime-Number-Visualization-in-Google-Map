Google Hadoop

The Magic of MapReduce: Map(), Shuffle(), Reduce() inspired by functional programming. Distributed System parallelly run tasks in mutiply nodes to facitate the computing. Strongly recommend Advanced Programming Paradigms(APP) course in year 3 taught by Dr. Brad.


Map(): Using Suburb name as Keyword, each year's crime number or unemployment rate grouped in an array as value.

{Suburb Name: [Start Year, Start Year + 1, ... , End Year]}

Shuffle(): Split a big task to severl small tasks and assign each small task to a slave node to computing in parallel


Reduce(): Each node to sum up own small task (suburb as key, summrize the same year crime number) to get a partial result, pass to master node and then master node combine them together to get final result
